## Installation

    git clone git://github.com/tobser/mythmpc.git
    cd mythmpc
    ./configure
    make
    sudo make install

## Configuration
you may simply add a button to your menu (see the [Menu theme development
guide](http://www.mythtv.org/wiki/Menu_theme_development_guide))

    <button>
       <text>mpc</text>
       <action>JUMP mpc</action>
    </button>
